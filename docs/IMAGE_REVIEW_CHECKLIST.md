# 学習補助画像レビューチェックリスト

## 使用方法

このチェックリストを使用して、日本語学習アプリの補助教材として使用する画像を厳しくレビューしてください。

---

## チェック項目

### 1. 意味と一致しているか

- [ ] 画像がことわざ・言い回しの意味と完全に一致しているか
- [ ] 画像を見て、ことわざの意味が理解できるか
- [ ] 意味とズレる要素が含まれていないか
- [ ] 間接的・連想的な表現ではなく、直接的な表現か

**チェック方法**:
1. ことわざの意味を確認
2. 画像に含まれる要素を列挙
3. 各要素が意味と一致しているか確認
4. 意味とズレる要素がないか確認

**不適切な例**:
- 「猿も木から落ちる」の画像に人間の失敗シーンが含まれている
- 「七転び八起き」の画像に階段を上る様子が含まれている（抽象的）

---

### 2. 初学者が誤解しないか

- [ ] 初学者が見て、正しい意味を理解できるか
- [ ] 誤解を招く要素が含まれていないか
- [ ] 文化的背景の知識がなくても理解できるか
- [ ] 画像だけで方向性が分かるか

**チェック方法**:
1. 初学者の視点で画像を見る
2. 画像から読み取れる情報を列挙
3. その情報が正しい意味に導くか確認
4. 誤解を招く可能性のある要素がないか確認

**不適切な例**:
- 「猿も木から落ちる」の画像で、猿が不器用に見える（「猿は木登りが苦手」と誤解される）
- 「継続は力なり」の画像で、特定の職業や状況が強調されている（意味が限定される）

---

### 3. 余計な情報が含まれていないか

- [ ] 学習に不要な装飾的要素が含まれていないか
- [ ] 背景がシンプル（白 or 単色）か
- [ ] 感情表現が過度に強調されていないか
- [ ] 特定の職業・状況・文化が強調されていないか

**チェック方法**:
1. 画像の各要素を列挙
2. 各要素が学習に必要か確認
3. 不要な要素を特定
4. 装飾的要素がないか確認

**不適切な例**:
- カラフルな背景や装飾的な効果
- 感情表現の強調（笑顔、悲しみなど）
- 特定の職業や状況（ビジネス、スポーツなど）

---

### 4. 文章説明なしでも方向性が分かるか

- [ ] 画像だけで、ことわざの意味の方向性が分かるか
- [ ] 画像を見て、正しい意味に導かれるか
- [ ] 説明文がなくても、誤解しないか

**チェック方法**:
1. 説明文を隠して画像だけを見る
2. 画像から読み取れる情報を列挙
3. その情報が正しい意味に導くか確認
4. 誤解を招く可能性がないか確認

**不適切な例**:
- 画像だけでは意味が分からない
- 画像から誤った意味が読み取れる
- 抽象的すぎて方向性が分からない

---

## 禁止表現チェック

以下の表現が含まれていないか確認:

- [ ] **かわいい**: 可愛らしい表現が含まれていないか
- [ ] **キャラクター**: アニメ風、マンガ風の表現が含まれていないか
- [ ] **デフォルメ**: デフォルメされた表現が含まれていないか
- [ ] **ファンタジー**: ファンタジー要素が含まれていないか
- [ ] **面白おかしい**: 面白おかしい表現が含まれていないか
- [ ] **抽象的**: 抽象的な表現が含まれていないか
- [ ] **メタファーのみ**: メタファーのみで表現されていないか

---

## 必須要件チェック

以下の要件を満たしているか確認:

- [ ] **現実に近い状況**: 現実に近い状況を描写しているか
- [ ] **実写写真 or 現実的なイラスト**: 実写写真または現実的なイラストか
- [ ] **装飾的要素なし**: 装飾的要素が一切含まれていないか

---

## レビューフォーマット

画像をレビューする際は、以下のフォーマットを使用してください:

```
【ことわざ】: [ことわざ名]
【意味】: [意味]

【画像レビュー】

1. 意味と一致しているか
   - 評価: [適切 / 不適切]
   - 理由: [理由]
   - 改善案: [改善案（不適切な場合）]

2. 初学者が誤解しないか
   - 評価: [適切 / 不適切]
   - 理由: [理由]
   - 改善案: [改善案（不適切な場合）]

3. 余計な情報が含まれていないか
   - 評価: [適切 / 不適切]
   - 理由: [理由]
   - 改善案: [改善案（不適切な場合）]

4. 文章説明なしでも方向性が分かるか
   - 評価: [適切 / 不適切]
   - 理由: [理由]
   - 改善案: [改善案（不適切な場合）]

【総合評価】: [適切 / 不適切]
【推奨アクション】: [使用可 / 改善が必要 / 使用不可]
```

## レビュー例

### 例1: 適切な画像

```
【ことわざ】: 猿も木から落ちる
【意味】: どんなに得意な人でも失敗することがある

【画像レビュー】

1. 意味と一致しているか
   - 評価: 適切
   - 理由: 猿が木に登っている様子（得意）と落ちた後の様子（失敗）が明確に表現されている

2. 初学者が誤解しないか
   - 評価: 適切
   - 理由: 猿が木登りが得意であることが明確で、それでも落ちることがあるという意味が伝わる

3. 余計な情報が含まれていないか
   - 評価: 適切
   - 理由: 背景はシンプルで、装飾的要素は一切ない

4. 文章説明なしでも方向性が分かるか
   - 評価: 適切
   - 理由: 画像だけで「得意なことでも失敗することがある」という意味の方向性が分かる

【総合評価】: 適切
【推奨アクション】: 使用可
```

### 例2: 不適切な画像

```
【ことわざ】: 猿も木から落ちる
【意味】: どんなに得意な人でも失敗することがある

【画像レビュー】

1. 意味と一致しているか
   - 評価: 不適切
   - 理由: アニメ風の可愛い猿が描かれており、現実に近い状況ではない

2. 初学者が誤解しないか
   - 評価: 不適切
   - 理由: 可愛い表現により、ことわざの真意（失敗は自然なこと）が伝わらない可能性がある

3. 余計な情報が含まれていないか
   - 評価: 不適切
   - 理由: カラフルな背景や装飾的要素が含まれている

4. 文章説明なしでも方向性が分かるか
   - 評価: 不適切
   - 理由: 可愛い表現により、学習者が「面白い」という印象を持ち、真意が伝わらない

【総合評価】: 不適切
【推奨アクション】: 改善が必要
【改善案】:
- アニメ風の表現をやめ、実写写真または現実的なイラストに変更
- 背景を白または単色に変更
- 装飾的要素を削除
- 猿が木登りが得意であることが明確に分かる表現に変更
```

---

## まとめ

画像をレビューする際は、以下の4つのチェック項目を厳しく確認してください:

1. **意味と一致しているか**
2. **初学者が誤解しないか**
3. **余計な情報が含まれていないか**
4. **文章説明なしでも方向性が分かるか**

さらに、禁止表現と必須要件も確認してください。

レビュー結果に基づいて、適切な画像を使用するか、改善案に従って修正してください。
