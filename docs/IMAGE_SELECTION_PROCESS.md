# 教材用画像選定プロセス

## 概要

日本語学習アプリの教材用画像を選定・使用するためのプロセスです。
画像は「正確さ」と「学習補助」を最優先に選定します。

---

## 選定プロセス（必須）

### ステップ1: 言葉の「具体的な意味」を文章で整理

ことわざ・言い回しの意味を、具体的で明確な文章で整理します。

**例**:
- 「猿も木から落ちる」→ 「猿は木登りが得意な動物だが、それでも落ちることがある。これは、どんなに得意なことでも失敗することがあるという意味を表す。」

### ステップ2: その意味を表す「現実世界の状況」を定義

意味を表す現実世界の具体的な状況を定義します。

**例**:
- 「猿も木から落ちる」→ 「自然環境で、猿が木に登っている様子、または木から落ちた後の様子」

### ステップ3: 検索に使う英語キーワードを複数提示

画像検索に使用する英語キーワードを複数提示します。

**例**:
- 「猿も木から落ちる」→ 
  - `japanese macaque tree climbing`
  - `monkey falling from tree`
  - `wild monkey in forest`
  - `primate tree natural habitat`

### ステップ4: 最適な画像を1つ選び、理由を説明

選定した画像とその理由を説明します。

**例**:
- 選定画像: [画像URL]
- 理由: 「猿が木に登っている様子と落ちた後の様子が明確に表現されている。背景はシンプルで、装飾的要素がない。現実に近い状況を描写している。」

### ステップ5: なぜ他の画像を選ばなかったかも簡潔に述べる

選ばなかった画像の理由を簡潔に述べます。

**例**:
- 「可愛い表現の画像は、ことわざの真意が伝わらないため除外」
- 「アニメ風のイラストは、現実に近い状況ではないため除外」
- 「装飾的な背景の画像は、学習に不要な情報が含まれるため除外」

## 画像取得ルール

### 使用可能なソース

- **Unsplash** (https://unsplash.com)
- **Pexels** (https://www.pexels.com)
- **Pixabay** (https://pixabay.com)
- その他、商用利用可能でライセンスが明確なフリー素材サイト

### ライセンス要件

- 商用利用可能であること
- ライセンスが明確であること
- クレジット表記が必要な場合は、適切に表記すること

### 画像タイプの優先順位

1. **実写写真**（最優先）
2. シンプルな説明イラスト（実写写真が適切でない場合のみ）

---

## 画像選定ルール

### ✅ 必須要件

1. **単語・意味と1対1で対応**: 画像が意味と完全に一致している
2. **現実世界の状況**: 現実に近い状況を描写している
3. **シンプルな背景**: 白または単色の背景
4. **装飾的要素なし**: 装飾的要素が一切含まれていない

### ❌ 禁止事項

1. **抽象的・比喩的・演出的な画像**: 概念を視覚化したものは不可
2. **キャラクター化・デフォルメ**: アニメ風、マンガ風の表現は不可
3. **誤解を招く構図**: 学習者が誤解しそうな構図は避ける
4. **装飾的要素**: 可愛い、面白い、かっこいいなどの装飾は不要

## 選定例: 「猿も木から落ちる」

### ステップ1: 言葉の「具体的な意味」を文章で整理

「猿も木から落ちる」は、「猿は木登りが得意な動物だが、それでも落ちることがある」という事実を通じて、「どんなに得意なことでも失敗することがある」という意味を表す。

### ステップ2: その意味を表す「現実世界の状況」を定義

- 自然環境で、猿が木に登っている様子（得意なこと）
- 猿が木から落ちた後の様子（失敗）
- 猿は冷静に状況を把握し、再び立ち上がろうとしている（失敗→学び）

### ステップ3: 検索に使う英語キーワードを複数提示

1. `japanese macaque tree climbing`
2. `monkey falling from tree natural`
3. `wild monkey forest tree`
4. `primate tree climbing realistic`
5. `monkey tree habitat documentary`

### ステップ4: 最適な画像を1つ選び、理由を説明

**選定画像**: [Unsplash等から選定]
**理由**: 
- 猿が木に登っている様子と落ちた後の様子が明確に表現されている
- 背景はシンプルで、装飾的要素がない
- 現実に近い状況を描写している
- 猿は冷静に状況を把握している様子が分かる

### ステップ5: なぜ他の画像を選ばなかったかも簡潔に述べる

- **可愛い表現の画像**: ことわざの真意（失敗は自然なこと）が伝わらないため除外
- **アニメ風のイラスト**: 現実に近い状況ではないため除外
- **装飾的な背景の画像**: 学習に不要な情報が含まれるため除外
- **猿が不器用に見える画像**: 「猿は木登りが得意」という前提が伝わらないため除外

## UI実装要件

### 画像表示

- 画像はカードUI内に表示
- レスポンシブ対応（max-width: 100%）
- 適切なアスペクト比を維持

### アクセシビリティ

- `alt`属性には必ず意味説明を書く
- 例: `alt="猿も木から落ちるの学習補助画像: どんなに得意な人でも失敗することがある"`

### 補足テキスト

- 画像下に補足テキストを表示
- フォーマット: 「この画像は◯◯を表しています。◯◯という意味を視覚的に理解するためのものです。」

---

## まとめ

画像選定は以下の5ステップで行います:

1. 言葉の「具体的な意味」を文章で整理
2. その意味を表す「現実世界の状況」を定義
3. 検索に使う英語キーワードを複数提示
4. 最適な画像を1つ選び、理由を説明
5. なぜ他の画像を選ばなかったかも簡潔に述べる

画像は必ず「正確さ」と「学習補助」を最優先に選定し、フリー素材のみを使用します。
