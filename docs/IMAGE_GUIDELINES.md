# 学習補助画像ガイドライン

## 基本方針

学習補助画像は、**学習者の理解を助けるための正確な視覚情報**を提供することを目的とします。
雰囲気づくりや装飾的な要素は一切含めません。

---

## 最重要ルール

### ✅ 必須

1. **正確性を最優先**: 意味と完全に一致する画像のみ使用
2. **実写写真 or シンプルな説明イラスト**: 抽象的・比喩的表現は禁止
3. **背景は白 or 単色**: 不要な装飾は一切しない
4. **一目で意味が分かる構図**: 複雑な構図は避ける

### ❌ 禁止事項（絶対に使用しない表現）

**以下の表現は画像生成・選定時に使用禁止です:**

1. **かわいい**: 可愛らしい表現は一切禁止
2. **キャラクター**: アニメ風、マンガ風、キャラクター化された表現は不可
3. **デフォルメ**: デフォルメされた表現は不可
4. **ファンタジー**: ファンタジー要素は一切禁止
5. **面白おかしい**: 面白おかしい表現は禁止
6. **抽象的**: 抽象的な表現は禁止
7. **メタファーのみで表現**: メタファー（比喩）のみで表現することは禁止

**最重要ルール**: 画像は必ず「現実に近い状況」を描写してください。

### ❌ その他の禁止事項

1. **装飾的要素**: 可愛い、面白い、かっこいいなどの装飾は不要
2. **意味とズレる画像**: 間接的・連想的な画像は不可

## 画像選定プロセス

### ステップ1: 画像の目的を明確化

**必ず以下を明示する:**

1. **画像の目的**: 何を理解させるか
2. **含めるべき要素**: 画像に必ず含めるべき具体的な要素
3. **絶対に含めてはいけない要素**: 誤解を招く可能性のある要素

### ステップ2: 意味と画像の対応関係を文書化

画像を選定・生成する前に、必ず文章で説明する:

```
【ことわざ】: 七転び八起き
【意味】: 何度失敗しても、あきらめずに立ち上がること

【画像の目的】:
「転倒して立ち上がる動作」を視覚的に理解させる

【含めるべき要素】:
- 人が転倒している様子
- 同じ人が立ち上がろうとしている様子
- 複数回の転倒と立ち上がりを示す連続的な動作（可能であれば）

【絶対に含めてはいけない要素】:
- キャラクターやアニメ風のイラスト
- 抽象的・比喩的な表現（例: 階段を上る、山を登るなど）
- 装飾的な背景や効果
- 感情表現（笑顔、悲しみなど）
- 特定の職業や状況（ビジネス、スポーツなど）

【推奨画像】:
- 実写写真: シルエットまたは後ろ姿で、転倒→立ち上がりの連続動作
- または、シンプルな線画イラスト: 人のシルエットが転倒し、立ち上がる様子
```

### 例2: 石の上にも三年

**意味**: 辛くても辛抱強く続ければ、いつか成果が出るということ

**画像の目的**: 「石の上に座り続ける動作」を視覚的に理解させる

**含めるべき要素**:
- 石（または岩）
- 人が石の上に座っている様子
- 長時間座り続けている様子（可能であれば時間の経過を示す）

**絶対に含めてはいけない要素**:
- キャラクターやアニメ風のイラスト
- 抽象的・比喩的な表現（例: 時計、カレンダーなど）
- 装飾的な背景や効果
- 感情表現（辛そう、我慢しているなど）
- 特定の職業や状況（修行、瞑想など）

**推奨画像**: 
- 実写写真: 白背景または自然背景で、石の上に座る人のシルエット
- または、シンプルな線画イラスト: 石と人のシルエット

---

### 例3: 猿も木から落ちる

**意味**: どんな名人や達人でも、時には失敗することがあるということ

**画像の目的**: 「猿が木から落ちる様子」を視覚的に理解させる

**含めるべき要素**:
- 猿（実物の猿）
- 木
- 猿が木から落ちている、または落ちた瞬間の様子

**絶対に含めてはいけない要素**:
- キャラクターやアニメ風のイラスト
- 抽象的・比喩的な表現（例: 人間が失敗している様子など）
- 装飾的な背景や効果
- 感情表現（驚き、悲しみなど）
- 人間の失敗シーン（意味がズレる）

**推奨画像**: 
- 実写写真: 自然環境で、猿が木から落ちる瞬間または落ちた後の様子
- または、シンプルな線画イラスト: 猿と木のシルエット、猿が落下している様子

### 例4: 継続は力なり

**意味**: 物事を続けていくことで、大きな力になるということ

**画像の目的**: 「継続的な動作」を視覚的に理解させる

**含めるべき要素**:
- 同じ動作を繰り返している様子（例: 毎日の運動、勉強など）
- 時間の経過を示す要素（可能であれば）

**絶対に含めてはいけない要素**:
- キャラクターやアニメ風のイラスト
- 抽象的・比喩的な表現（例: 成長の階段、光など）
- 装飾的な背景や効果
- 感情表現（頑張っている、達成感など）
- 特定の職業や状況（アスリート、学生など）

**推奨画像**: 
- 実写写真: 白背景で、同じ動作を繰り返す人のシルエット（連続写真風）
- または、シンプルな線画イラスト: 同じ動作を繰り返す人のシルエット

---

### 例5: 急がば回れ

**意味**: 急いでいる時こそ、安全で確実な方法を選ぶべきだということ

**画像の目的**: 「近道と遠回り」を視覚的に理解させる

**含めるべき要素**:
- 2つの道（近道と遠回り）
- 近道が危険または困難である様子
- 遠回りが安全である様子

**絶対に含めてはいけない要素**:
- キャラクターやアニメ風のイラスト
- 抽象的・比喩的な表現（例: 時計、矢印など）
- 装飾的な背景や効果
- 感情表現（焦り、安心など）
- 特定の職業や状況（ビジネス、スポーツなど）

**推奨画像**: 
- 実写写真: 地図または実際の道の写真（近道が険しい、遠回りが平坦）
- または、シンプルな線画イラスト: 2つの道を示す地図風の図

## 技術実装

### 画像の保存方法

1. **ローカル画像**: `public/images/phrases/` に保存
2. **外部画像**: URLを指定（Unsplash、Pexelsなどの実写写真サイト推奨）
3. **生成画像**: AI画像生成を使用する場合は、必ず上記ガイドラインに従う

### 画像の命名規則

```
phrase-{id}-{description}.jpg
例: phrase-1-fall-and-rise.jpg
```

### 画像のサイズ

- **推奨サイズ**: 800x600px または 1200x900px
- **アスペクト比**: 4:3 または 16:9
- **ファイル形式**: JPG（実写）、PNG（イラスト）

## まとめ

学習補助画像は、**正確さを最優先**に選定・生成します。

### 絶対に使用禁止の表現

以下の表現は画像生成・選定時に**絶対に使用しない**でください:

- **かわいい**
- **キャラクター**
- **デフォルメ**
- **ファンタジー**
- **面白おかしい**
- **抽象的**
- **メタファーのみで表現**

### 必須要件

画像は必ず**「現実に近い状況」を描写**してください。

### 画像選定・生成の手順

画像選定・生成の際は、必ず以下を明示してください:

1. **画像の目的**: 何を理解させるか
2. **含めるべき要素**: 画像に必ず含めるべき具体的な要素（現実に近い状況）
3. **絶対に含めてはいけない要素**: 誤解を招く可能性のある要素（上記禁止表現を含む）

そして、**意味と画像の対応関係を文章で説明**してから、UIに組み込んでください。
